# Product Scope

## Overview

Rallies is a multilingual web platform for table tennis players, clubs, and event managers.

It combines identity, community, discovery, match tracking, and monetization in a single platform focused on the table tennis ecosystem.

## Core audiences

- Players
- Clubs
- Event organizers
- Platform administrators

## Main product areas

### Authentication and account access
Authentication may be delegated to a third-party provider. The platform still owns application-level users, roles, profile completion state, permissions, and business identity.

### Public player profiles
Each player can have a public profile page with sport-related metadata, visibility preferences, and localization-aware presentation.

### Clubs
Clubs work like groups or clans:
- membership
- invitations or join flows
- club roles
- club page
- future chat or announcements

### Events
Event managers can create sport-related events and publish them for discovery. Users can browse and filter events by location, name, and other metadata.

### Scoreboard
The scoreboard supports:
- interactive point tracking
- private sessions
- broadcast mode
- match history persistence
- 2-player or 4-player match association

### Contact and support
Users can submit support, suggestion, or feedback messages. These can be handled through email or an internal admin-managed workflow.

### Credits and payments
Users can purchase credits and use them for premium actions such as:
- creating a club
- creating an event
- boosting an event
- expanding scoreboard broadcast reach

Subscriptions may offer better cost-benefit and recurring credit allocation.

## Non-functional goals

- Mobile-first
- PWA-ready
- High performance
- Security-first
- Automation-friendly
- Suitable for AI-assisted development

## Multilingual support

Initial locales:
- English (`en`)
- Portuguese, Brazil (`pt-BR`)
